Understanding Color Models and Spot Color SystemsIC

A color model is a system for creating a full range of colours from a small set of primary colors. There are two types of colour models: additive and subtractive. Additive color models use light to display color, while subtractive color models use printing inks. The most common color models that graphic designers work with are the CMYK model for printing and the RGB model for computer display. Read more...
